# 모바일

Marketo Mobile Engagement는 마케터가 모바일 경험을 제공하고 모바일 디바이스에서 고객 활동에 응답할 수 있도록 지원합니다.

Marketo Mobile Engagement에는 두 가지 구성 요소가 있습니다.

1. Mobile SDK

   Marketo Mobile SDK을 모바일 애플리케이션에 통합하여 마케팅 콘텐츠를 제공하고 분석을 수집합니다.

1. 모바일 마케팅

   Marketo을 사용하여 대상을 정의하고, 알림을 만들고, 타겟팅된 메시징 캠페인을 시작하고, 고객 분석을 볼 수 있습니다.

Marketo SDK을 모바일 애플리케이션에 통합하면 Marketo에서 모바일 푸시 알림과 인앱 메시지를 전송할 수 있습니다. 모바일 메시지를 설정하고 배포하려면 제품 설명서에서 [모바일 마케팅](https://experienceleague.adobe.com/en/docs/marketo/using/product-docs/mobile-marketing/admin/add-a-mobile-app)을 참조하세요.
